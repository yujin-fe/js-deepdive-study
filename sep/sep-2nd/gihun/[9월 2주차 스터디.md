# 객체 

##  가능한 값

+ 객체(참조) 타입의 값은 변경 가능한 값

+ 메모리 주소를 통해 접근하여 원시 값 자체를 값으로 가짐 

+ 메모리 공간 주소의 접근 하는것을 참조 값 이라고 함

## 얉은 복사 깊은 복사 

```
const a = { x: { y : 1 } }

const c1 = {... a };
console.log (c1 === a );
console.log( c1.x === a.x ) 

```

+ 바로 안쪽 속성 만 새 객체에 복사 

+ 객체나 배열은 참조만 복사됨 

+ 내부 객체를 다른 변수에서 수정시 원본도 영향을 받음 

```
const a = { x: { y: 1 } };
const c1 = { ...a };

c1.x.y = 999;

console.log(a.x.y); // 999 (원본도 같이 바뀜!)

```

 ## 깊은 복사 

 + 보여지는 갚으로는 똑같지만 

 + 내부적으로 메모리 주소가 다름 

 + 그리하여 일치 비교를 했을때 false 값이 출력됨
